## 開発環境構築
### 事前準備
* Python 3のインストール
* Discordボットのトークンの取得

### 構築方法
1. これレポジトリを`git clone`する
2. レポジトリのクローン先へ移動
3. Python仮想環境の作成・アクティベート
4. 必要なモジュールをインストール
```shell
pip install -r requirements.txt
```
5. `dchanbot`ディレクトリへ移動
6. `.env`ファイルの作成
7. `.env`ファイルを次のように編集
```
DBOT_TOKEN = "取得したDiscordボットのトークン"
```
8. ボットを実行するには`dchanbot`ディレクトリ直下の`main.py`を実行してください
